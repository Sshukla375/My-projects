Unix Shell (C++)
A simple Linux-based shell built in C++.

 Features:
- Command execution
- Piping support
- Input/output redirection
- Process management using fork(), exec(), wait()

This project demonstrates basic operating system concepts and system-level programming.




A* Pathfinding Project (C++)
Implements the A* algorithm to find the shortest path in a grid.

Features:
- Graph-based pathfinding
- Heuristic-based optimization
- Visualization of shortest path logic

This project demonstrates graph algorithms and optimization techniques.
